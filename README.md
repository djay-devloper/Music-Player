🎵 Music Player Web App

A simple and responsive Music Player built with HTML, CSS, and JavaScript.
It allows users to play songs, pause music, switch between tracks, and view the song progress with a dynamic progress bar.




🚀 Features

▶️ Play and Pause music

⏭️ Next and Previous track controls

🎧 Playlist support using JavaScript array

⏱️ Current time and total duration display

📊 Interactive progress bar

🔁 Auto play next track when the current song ends

🖼️ Dynamic song title, artist name, and cover image updates

🎨 Clean and responsive UI design




🛠️ Technologies Used

HTML5 – Structure of the music player

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Player functionality and logic

Font Awesome – Icons for controls
